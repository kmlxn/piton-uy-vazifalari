```
xolodilkin.zasunut('xleb', 2)
xolodilkin.zasunut('frukti', 21)
xolodilkin.zasunut('soki', 4)

# 1. riba - 5 litr
# 2. xleb - 2 litr
# 3. frukti - 21 litr
# 4. soki - 4 litr
# Obshiy obyem produktov 28 litr
```
